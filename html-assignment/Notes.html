The five languages are

1.<p></p>:this tag is used to define or show a paragraph of text. Browsers typically adds some space below and above the paragraph. 2.<div></div>:this tag is used for styling purposes and to create sections within a web page. I think it is mainly used for organization. 3.<img></img>: this tag is used to attach images into an HTML document. And this tag doesn’t have a closing tag like others. It is followed by ‘src ’ which explains the source of the image.4.<ul></ul>: this tag is used to create unordered list in HTML. 5.<body></body>: this tag is used to define the main content in the HTML document which may include text, images, links and others.
